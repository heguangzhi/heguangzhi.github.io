在CentOS 6.X 上面安装 Python 2.7.X

作者： Frank.he 分类： Python    时间：2016-08-05 10:20

CentOS 6.X 自带的python版本是 2.6 , 由于工作需要，很多时候需要2.7版本。所以需要进行版本升级。由于一些系统工具和服务是对 Python 有依赖的，所以升级 Python 版本需要注意。

升级步骤

源码安装Python 2.7.x

cd /usr/local/src
wget https://www.python.org/ftp/python/2.7.12/Python-2.7.12.tgz
tar -xzvf Python-2.7.12.tgz

安装详情：

# 进入目录:
cd Python-2.7.12
# 运行配置 configure:
./configure --prefix=/usr/local
# 编译安装:
make
make altinstall
# 检查 Python 版本:
[root@localhost ~]# python2.7 -V
Python 2.7.12

设置 PATH

为了方便使用Python，设置系统变量或者建立 软连接将新版本的 Python加入到 path 对应的目录 ：

#先删除系统原来的软连接
[root@localhost ~]#rm /usr/bin/python
export PATH="/usr/local/bin:$PATH"
or 
ln -s /usr/local/bin/python2.7  /usr/bin/python
# 检查
[root@localhost ~]# python -V
Python 2.7.12
[root@localhost ~]# which python 
/usr/bin/python

安装 setuptools

setuptools是Python distutils增强版的集合，它可以帮助我们更简单的创建和分发Python包，尤其是拥有依赖关系的。用户在使用setuptools创建的包时，并不需要已安装setuptools，只要一个启动模块即可。

#获取软件包
cd /usr/local/src
wget --no-check-certificate https://pypi.python.org/packages/source/s/setuptools/setuptools-1.4.2.tar.gz
# 解压:
tar -xvf setuptools-1.4.2.tar.gz
cd setuptools-1.4.2
# 使用 Python 2.7.12 安装 setuptools
python2.7 setup.py install

安装 PIP

pip是一个安装和管理 Python 包的工具

curl  https://bootstrap.pypa.io/get-pip.py | python2.7 -

修复 yum 工具

此时yum应该是失效的，因为此时默认python版本已经是2.7了。而yum需要的是2.6 所以：

[root@localhost ~]# which yum 
/usr/bin/yum
#修改 yum中的python 
将第一行  #!/usr/bin/python  改为 #!/usr/bin/python2.6
此时yum就可以正常使用了

总结

之前想用yum直接安装python2.7，其实用源码安装跟方便。
