Pelican搭建静态博客

作者： Frank.he 分类： Pelican    时间：2016-08-12 22:10

安装Pelican需要本机安装Python 2.7.x或者Python 3.3+，同时需要安装pip，执行下面的命令，如果在linux有权限限制就在命令前面加一个sudo，写博客用Markdown，所以要安装markdown模块.

安装Python 2.7.x 请参考 《在CentOS 6.X 上面安装 Python 2.7.X》

Pelican voidy-bootstrap 加载慢的解决方案

主要原因是加载CSS，字体，而国内的网络环境又不允许。解决方案就是找到国内的开放公共库。 国内开源项目CDN服务 http://www.bootcdn.cn/

修改base.html 如下：

https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css
替换为
//cdn.bootcss.com/bootstrap/3.3.5/css/bootstrap.min.css

//libs.baidu.com/fontawesome/4.0.3/css/font-awesome.min.css
替换为
//maxcdn.bootstrapcdn.com/font-awesome/4.4.0/css/font-awesome.min.css

      <!--   <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css" /> -->

添加多说评论

disqus是pelican默认支持的评论系统，由于访问的速度问题，选择国内的多说：

首先在多说的网站中注册一个账号. 填写相关信息得到如下的代码：

<!-- 多说评论框 start -->
    <div class="ds-thread" data-thread-key="请将此处替换成文章在你的站点中的ID" data-title="请替换成文章的标题" data-url="请替换成文章的网址"></div>
<!-- 多说评论框 end -->
<!-- 多说公共JS代码 start (一个网页只需插入一次) -->
<script type="text/javascript">
var duoshuoQuery = {short_name:"code-doc"};
    (function() {
        var ds = document.createElement('script');
        ds.type = 'text/javascript';ds.async = true;
        ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
        ds.charset = 'UTF-8';
        (document.getElementsByTagName('head')[0] 
         || document.getElementsByTagName('body')[0]).appendChild(ds);
    })();
    </script>
<!-- 多说公共JS代码 end -->

需要替换以及修改部分内容，如：

{% if DUOSHUO_SITENAME %}
    <hr/>
    <section class="comments" id="comments">
        <h2>Comments</h2>
        <div class="ds-thread" data-thread-key="{{ article.slug }}" data-title="{{ article.title }}" data-url="{{ SITEURL }}/{{ article.url }}"></div>
        <script type="text/javascript">
            var duoshuoQuery = {short_name:"{{ DUOSHUO_SITENAME }}"};
            (function() {
                var ds = document.createElement('script');
                ds.type = 'text/javascript';ds.async = true;

                 || document.getElementsByTagName('body')[0]).appendChild(ds);
            })();
        </script>
        <noscript>Please enable JavaScript to view the <a href="http://duoshuo.com/">comments powered by Duoshuo.</a></noscript>
    </section>
{% endif %}

修改模板文件

修改article.html内容, 在</article>内,最后一个endif之后添加如下内容

    <hr/>
    <section class="comments" id="comments">
        <h2>Comments</h2>
        <div class="ds-thread" data-thread-key="{{ article.slug }}" data-title="{{ article.title }}" data-url="{{ SITEURL }}/{{ article.url }}"></div>
        <script type="text/javascript">

修改配置文件

在Pelicanconf.py中添加

DUOSHUO_SITENAME = "code-doc"

然后重新生成网站就会看到相关的评论界面了.
