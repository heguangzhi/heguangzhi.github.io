安装MySQL Server 5.7.14 in CentOS 6.x and Red Hat 6.x Linux

作者： Frank.he 分类： MySQL    时间：2016-08-25 15:07

操作系统: CentOS 6.5 Arch: x86_64

从MySQL官方站点下载repo rpm 包.

登录/注册MySQL官网，下载yum repo rpm宝 (URL: http://dev.mysql.com/downloads/repo/) 或者直接从 http://repo.mysql.com 选择要下载的rpm包 

cd /usr/local/src
yum install wget
wget http://repo.mysql.com/mysql57-community-release-el6-8.noarch.rpm
rpm -ivh mysql57-community-release-el6-8.noarch.rpm

安装MySQL server

#yum install mysql-server

查看MySQL数据库的配置信息

MySQL的配置文件依然是/etc/my.cnf，其它安装信息可以通过mysql_config命令查看。其中，动态库文件存放在/usr/lib64/mysql目录下。

删除MySQL的Repository

安装了MySQL的Yum Repository，所以以后每次执行yum操作时，都会去检查更新。如果想要去掉这种自动检查操作的话，可以使用如下命令卸载MySQL的Repository即可。

[root@localhost ~]# yum -y remove mysql57-community-release-el6-8.noarch
