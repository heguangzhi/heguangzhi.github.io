通过Linux ISO安装文件通过yum进行本地库安装

作者： Frank.he 分类： Linux    时间：2016-08-15 15:07

系统管理员帮助安装了CentOS6.5系统，发现是最小化安装的。没有安装开发工具包。采取的方案是从网上下载CentOS-6.5 iso文件外挂到系统上，通过ISO文件进行yum 安装开发工具包。

挂载ISO文件

上传iso文件 /usr/local/src/ 通过mount 命令把iso文件外挂到/mnt/vcdrom 

mount -o loop -t iso9660 /usr/local/src/CentOS-6.5-x86_64-bin-DVD1.iso   /mnt/vcdrom

批量备份原来的YUM配置文件

# cd /etc/yum.repos.d
# rename .repo .repo.bak *.repo

编辑本地yum repo库

cd /etc/yum.repos.d
vim  CentOS-Media.repo 

[c6-media]
name=CentOS-$releasever - Media
baseurl=file:///mnt/vcdrom/
gpgcheck=1
enabled=1
gpgkey=file:///mnt/vcdrom/RPM-GPG-KEY-CentOS-6

安装开发包

yum groupinstall 'Development Tools'
